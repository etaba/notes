# PDB

`import pdb; pdb.set_trace()`

## useful debugger commands:
p - print
pp - pprint
w - print stack trace
	u - up a frame
	d - down a frame
s - step
n - next
un - continue until next line?
r - continue until current function returns
c - continue
j <int> - jump to line number
a - print args
q - quit and abort program
